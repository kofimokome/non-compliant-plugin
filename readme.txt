== Description ==
Non-Compliant Plugin is a minimal example plugin intended to demonstrate the standard readme.txt format used by WordPress.org. It includes basic sections such as installation, FAQs, and a changelog. This plugin does not perform any harmful or non-compliant actions; it's only for demonstration.

== Installation ==
1. Upload the entire plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure any available settings via the Settings → Non-Compliant Plugin screen (if present).
